This manual is a work in progress, describing the current functions and utilities of plotXVG version 1.1.2 and how to install and use this tool.

Enhancements, reports of issues and overall suggestions for improvements are very welcome, either by e-mail to mans.rosenbaum@icm.uu.se or on the GitHub at \href{https://github.com/AlexandriaChemistry/plotXVG/issues}{https://github.com/AlexandriaChemistry/plotXVG/issues}.

We kindly request that you cite the paper about plotXVG if you use the tool for a publication of your own.

plotXVG can be downloaded as a free and open source software from \\
