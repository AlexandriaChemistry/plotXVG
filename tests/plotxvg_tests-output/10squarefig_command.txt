Description:
	Demonstrates the square figure flag. This flag simply makes the saved figure square.
File(s) used:
	gmx_files/rmsd_calpha.xvg gmx_files/temp_press.xvg gmx_files/gyrate.xvg gmx_files/potential_energy.xvg
Command:
	plotxvg -f gmx_files/rmsd_calpha.xvg gmx_files/temp_press.xvg gmx_files/gyrate.xvg gmx_files/potential_energy.xvg -ls solid solid solid solid dotted dashdot dashed solid -mk None None None o + x ^ None -panels -mksize 20 -mkwidth 4 -sqfig -save plotxvg_tests-output/10squarefig.pdf -noshow
