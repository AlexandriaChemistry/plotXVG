Description:
Using the panels flag, along with -notitles. 
Also note -sharelabel, which removes axis labels except for the first column and the last row. Suitable if all subplots shares the same axis labels.

plotxvg -f gmx_files/rmsd_calpha.xvg gmx_files/rmsd_sidechain.xvg -panels -sharelabel -notitles -ls solid  -save plotxvg_tests-output/06two_panels.pdf -noshow
